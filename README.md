# Stock Portfolio Analysis

## Overview
This project analyses the performance of three stock portfolios using financial and risk-return metrics.

## Tools Used
- Excel (financial modelling, analysis)
- Data visualisation

## Key Analysis
- Compared portfolio returns and volatility
- Evaluated diversification effects
- Analysed risk-return trade-offs

## Key Insights
- Diversified portfolios showed reduced volatility
- Higher returns were associated with increased risk
- Portfolio composition significantly impacted performance
